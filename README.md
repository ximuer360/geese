# 项目资源分享平台

一个用于分享和管理项目资源的全栈应用，支持项目展示、标签分类、搜索功能等。

## 功能特点

### 前台功能
- [x] 项目列表展示
  - 支持分页加载
  - 卡片式布局
  - 封面图片展示
  - 项目基本信息（名称、描述、语言、Stars等）
  - 标签显示

- [x] 标签分类导航
  - 资源类型分类
  - 技术栈分类
  - 地区分类
  - 标签统计数量

- [x] 搜索功能
  - 支持项目名称搜索
  - 支持项目描述搜索
  - 支持编程语言搜索
  - 支持标签搜索

### 后台功能
- [x] 项目管理
  - 新增项目
  - 编辑项目
  - 删除项目
  - 项目封面上传
  - 多图片管理

- [x] 标签管理
  - 标签分类展示
  - 新增标签
  - 编辑标签
  - 删除标签
  - 标签统计

- [x] 系统功能
  - 登录验证
  - 权限控制
  - 系统设置

## 技术栈

### 前端
- Vue 3
- TypeScript
- Vue Router
- Tailwind CSS
- Axios
- ByteMD (Markdown 编辑器)

### 后端
- Express
- TypeScript
- Prisma ORM
- SQLite

## 本地开发

1. 克隆项目
```bash
git clone <repository-url>
```

2. 安装依赖
```bash
# 前端
cd frontend
npm install

# 后端
cd backend
npm install
```

3. 启动开发服务器
```bash
# 前端
npm run dev

# 后端
npm run dev
```

4. 访问
- 前台：http://localhost:5173
- 后台：http://localhost:5173/admin
- 后端API：http://localhost:5001

## 环境变量

后端 `.env` 文件配置：
```env
DATABASE_URL="file:./dev.db"
PORT=5001
ADMIN_PASSWORD="your_password"
```

## 部署

1. 构建前端
```bash
cd frontend
npm run build
```

2. 构建后端
```bash
cd backend
npm run build
```
